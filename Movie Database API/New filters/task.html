Description

As time flies, your database will grow. At some point, retrieving all the data may become problematic: the response will be too large, which will make it harder for the end client to process it, and gathering all the rows from the database will require time. This is where you should think about allowing your API to filter data your clients want to retrieve. Filtering is one of the popular REST concepts that makes your API is easier to use by clients. In simple cases, filtering is usually implemented with query parameters.

Objectives

Add the ability to filter Humans by role and Movies by genre to the corresponding GET /humans and GET /movies endpoints using query parameters. When receiving the query parameter, the GET endpoint should return only people or movies that stored in the database.

Read more on using query parameters for filtering and other clues in the REST API Design: Filtering, Sorting, and Pagination article by Moesif.

Examples

Example 1: returning multiple individuals with different roles

http GET http://127.0.0.1:8000/humans

example response:

{"humans":
     [{"id": 1,
       "name": "Terry Gilliam"}, # assuming he has following roles: ["actor", "director"]
      {"id": 2,
       "name": "Graham Chapman"}, # ["actor", "writer"]
      {"id": 3,
       "name": "John Cleese"}] # ["actor", "writer"] stored in the database

Example 2: getting people with a writer role

http GET http://127.0.0.1:8000/humans?role=writer

example response:

{"humans":
    [{"id": 2,
      "name": "Graham Chapman"},
     {"id": 3,
      "name": "John Cleese"}]

Example 3: returning multiple movies of different genres

http GET http://127.0.0.1:8000/movies

example response:

{"movies": [
     {"id": 1,
      "name": "Monthy Python's Flying Circus",
      "genre": "comedy"},
     {"id": 2,
      "name": "Hypermovie",
      "sci-fi"}
]}

Example 4: getting movies with a comedy genre

http GET http://127.0.0.1:8000/movies?genre=comedy

example response:

{"movies": [
     {"id": 1,
      "name": "Monthy Python's Flying Circus",
      "genre": "comedy"}
]}
